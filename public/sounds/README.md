# 音效檔案說明

此目錄存放遊戲所需的音效檔案。請準備以下 MP3 格式的音效：

## 必要音效

1. **click.mp3** - 點擊格子音效
   - 用於：Luck Board 點擊格子時
   - 建議：短促的點擊聲（如按鈕音效）

2. **hit.mp3** - 命中音效
   - 用於：Luck Board 抽中綠格（得分）
   - 建議：愉悅的成功音效（如叮咚聲、歡呼聲）

3. **miss.mp3** - 落空音效
   - 用於：Luck Board 抽中紅格（不得分）
   - 建議：失望音效（如嘆氣聲、咚的一聲）

4. **correct.mp3** - 答對音效
   - 用於：Quiz 答對題目時
   - 建議：正確提示音（如叮、鈴聲）

5. **reveal.mp3** - 揭曉音效
   - 用於：投票結果逐人揭曉時
   - 建議：懸念音效（如鼓聲、開箱聲）

6. **finish.mp3** - 結束音效
   - 用於：關卡完成、遊戲結束
   - 建議：勝利音效（如喇叭聲、掌聲）

## 音效規格建議

- 格式：MP3
- 時長：1-3 秒（建議短促）
- 音量：建議先正規化到 -3dB 至 -6dB
- 取樣率：44.1kHz 或 48kHz

## 臨時替代方案

如果暫時沒有音效檔案，可以：
1. 使用線上免費音效網站（如 freesound.org, zapsplat.com）
2. 使用文字轉語音工具產生簡單提示音
3. 暫時關閉音效功能（在管理頁面設定）

## 實作注意事項

音效由 `composables/useAudio.ts` 管理，支援：
- 預載所有音效（減少延遲）
- 音量控制
- 靜音開關
- 循環播放（如背景音樂）
