## 🧪 測試步驟

### 準備工作

1. **確認開發伺服器運行中**
   ```bash
   # 如果沒有運行，執行：
   npm run dev
   ```
   開啟 `http://localhost:3001`

2. **載入測試資料**
   - 前往首頁
   - 點擊「📦 載入假資料（測試用）」
   - 確認載入 5 位玩家

### 測試流程

#### 測試 1：基本抽取功能 ✅

1. 點擊首頁的「🎁 幸運盤面（新！）」按鈕
2. **預期**：進入 Luck Board 頁面，看到 10×6 盤面

3. 在左側玩家列表選擇「Alice」（或按數字鍵 `1`）
4. **預期**：Alice 被標記為「當前」玩家

5. 點擊盤面上任一格子
6. **預期**：
   - 格子翻轉動畫
   - 顯示禮物 🎁 或便便 💩
   - 如果是禮物：右側排行榜 Alice 分數 +1
   - 格子上顯示 Alice 名字與代表色
   - 該格子變暗無法再點擊

#### 測試 2：輪抽流程 ✅

1. 選擇 Bob（按 `2`）
2. 點擊另一格
3. **預期**：格子顯示 Bob 的名字與代表色

4. 重複選擇不同玩家並抽取
5. **預期**：每位玩家可以抽取任意格子

#### 測試 3：透視模式 ✅

1. 按 `Alt + V` 或點擊「👁️ 透視模式」按鈕
2. **預期**：
   - 未抽取的格子右上角顯示 🎁 或 💩
   - 可看到所有格子的真實內容

3. 再按一次 `Alt + V`
4. **預期**：透視提示消失

#### 測試 4：重新洗牌 ✅

1. 抽取 5-10 格
2. 點擊「🔄 重新洗牌」按鈕
3. 確認對話框，點擊「確定」
4. **預期**：
   - 所有格子重置為未抽取狀態
   - 盤面重新隨機生成
   - 分數保持不變（可選：重置分數）

#### 測試 5：排行榜 ✅

1. 讓不同玩家抽取不同數量的綠格
2. **預期**：
   - 右側排行榜依分數排序
   - 前三名顯示 🥇🥈🥉
   - 顯示每位玩家的綠格命中數

3. 點擊「🏆 排行榜」按鈕
4. **預期**：跳轉到排行榜頁面

#### 測試 6：統計資訊 ✅

1. 持續抽取格子
2. **預期**：
   - 頂部統計資訊即時更新
   - 進度條隨著抽取增長
   - 🎁 和 💩 計數正確

#### 測試 7：響應式設計 ✅

1. 調整瀏覽器視窗大小
2. **預期**：
   - 桌機版：三欄佈局（玩家/盤面/排行榜）
   - 平板/手機：單欄佈局，只顯示盤面
   - 格子大小自動調整

#### 測試 8：快捷鍵 ✅

1. 按數字鍵 `1` 到 `5`
2. **預期**：快速切換玩家

3. 按 `Alt + V`
4. **預期**：切換透視模式

## 💡 使用技巧

### 主持人操作建議

1. **開始前**
   - 使用透視模式檢查盤面分布
   - 確認玩家都已載入

2. **遊戲中**
   - 使用數字鍵快速切換玩家
   - 觀察右側排行榜掌握分數
   - 可隨時查看統計資訊

3. **遊戲結束**
   - 點擊「🏆 排行榜」查看最終排名
   - 可重新洗牌再玩一輪
   - 點擊「▶️ 下一關」進入下個遊戲

### 投影建議

- **解析度**：1920×1080 或以上
- **縮放**：可使用瀏覽器縮放（Ctrl/Cmd +/-）
- **全螢幕**：按 F11 進入全螢幕模式
- **隱藏左右面板**：視窗寬度 < 1200px 自動隱藏
